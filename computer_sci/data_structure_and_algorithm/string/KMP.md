# Details

## What's KMP do

KMP是做**字符串匹配**最常用的算法之一。

> [!abstract] 
> 什么是字符串匹配？ 
> 
> 举例来说，有一个字符串"BBC ABCDAB ABCDABCDABDE"，我想知道，里面是否包含另一个字符串"ABCDABD"？


Knuth-Morris-Pratt算法是以三个发明者命名，起头的那个K就是著名科学家Donald Knuth。
